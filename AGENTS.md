# Codex / Claude Code 项目执行规则

本工程是面向跨境高端电商腕表品类的 AI 自动化运营平台 Demo，后续目标是商用交付。

## 第一目标

所有工作以最终可交付结果为核心。

不要机械执行临时方案。用户方案是输入，不是唯一答案。如果发现更稳、更简单、更适合商用交付和小白团队维护的方案，必须直接提出。

## 执行原则

- 商用交付优先，不做花架子。
- 结果优先，不机械照做。
- 如果用户方案不合适，直接指出并提供更优方案。
- 可以推荐更合适的插件、平台、软件、架构或流程，但必须说明落地原因。
- 小白团队可维护优先。
- 不过度复杂化。

## AI 自动化边界

AI 可以做翻译、总结、回复草稿、低风险引导、信息收集、离线日报。

AI 禁止自动承诺退款、补偿、改价、取消订单、修改广告预算、发送高风险客户消息、绕过人工审核。

## 禁止事项

- 不写入真实 token、secret、邮箱密码。
- 不接个人微信自动化。
- 不自动发送真实客户消息。
- 不绕过人工审核。
- 不删除现有 Demo。
- 不未经确认修改 Git 远程配置。

## 提交前校验

至少运行：

```text
node --check src/app.js
node scripts/verify-template.mjs
git diff --check
```

如果依赖已安装，再运行：

```text
npm run typecheck
npm run lint
npm run build
```

如果依赖未安装，必须说明不能完成 TypeScript 完整校验。
