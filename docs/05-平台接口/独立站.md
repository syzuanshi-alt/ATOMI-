# 独立站接口

## 一期目标

- 独立站订单同步。
- 客户资料同步。
- 支付状态和订单状态同步。
- Webhook 预留。

## Shopify 优先能力

- Admin 接口（API）。
- Orders。
- Customers。
- Fulfillment。
- Webhooks。

## 设计原则

- 订单以 `orders.external_order_id` 去重。
- 客户以邮箱/手机号/外部 ID 做匹配。
- Webhook 和定时补偿同步并存。
