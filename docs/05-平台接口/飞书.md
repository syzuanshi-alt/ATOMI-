# 飞书接口

## 一期能力

- 多维表格记录写入。
- 多维表格记录更新。
- 飞书群消息推送。
- 日报、订单异常、内容复盘、任务同步。

## 准备项

- 飞书自建应用。
- App ID。
- App Secret。
- 多维表格 token。
- 表 ID。
- 群 chat ID。
- 应用权限授权。

## 设计原则

- 飞书不是主数据库。
- 飞书写入必须幂等。
- 同步失败要写 `sync_runs` 和 `feishu_exports`。
