# 权限模型

## 角色

- 平台管理员。
- 客户管理员。
- 老板只读。
- 运营。
- 客服。
- BD。
- 投手。

## 权限原则

- 平台管理员可管理所有租户配置，但不默认参与客户业务操作。
- 客户管理员只管理本租户成员和配置。
- 运营查看内容数据和任务。
- 客服查看客户、订单、物流和会话。
- BD 查看达人和素材授权。
- 老板只读查看日报、看板和异常。

## 验收

- 客户 A 看不到客户 B 数据。
- 非授权角色不能导出敏感数据。
- 权限变更写入审计日志。
