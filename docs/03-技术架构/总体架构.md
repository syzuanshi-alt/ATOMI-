# 总体架构

## 架构分层

1. 数据接入层：抖音、抖店、独立站、TikTok/TK、飞书、邮件、物流。
2. 数据中台层：客户、订单、内容、素材、达人、任务、AI 输出。
3. AI 分析层：日报、复盘、建议、异常识别。
4. 自动流程层：同步任务、飞书输出、邮件草稿、任务生成。
5. 操作端：电脑端后台、手机端 H5/PWA。
6. 协同出口：飞书多维表格、飞书群、邮件通知。

## 技术默认

- Next.js App Router。
- React + TypeScript。
- PostgreSQL。
- Prisma。
- GitHub Projects。
- Obsidian Markdown。
- 飞书 OpenAPI。

## 核心原则

- 多租户隔离优先。
- 数据库是主数据源。
- 飞书是协同出口。
- AI 输出可追溯。
- 高风险自动化需要人工审核。
