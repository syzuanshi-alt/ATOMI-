# 平台总控

## 目标

平台方通过总控后台管理客户、权限、模块、授权、同步状态和交付范围。

## 一期功能

- 客户公司列表。
- 创建客户公司。
- 开通/关闭模块。
- 查看平台授权状态。
- 查看同步任务状态。
- 查看接口错误和同步日志。
- 查看成员和角色。
- 查看客户可见后台入口。

## 验收标准

- 平台方能创建至少 2 个客户公司。
- 客户之间数据隔离。
- 模块开关生效。
- 同步失败能看到错误原因。
