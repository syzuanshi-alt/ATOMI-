# AI 日报

## 目标

每天自动汇总经营数据、视频数据、订单异常、客服问题、达人跟进和今日重点任务。

## 输入

- 内容数据。
- 订单数据。
- 物流数据。
- 客服消息。
- 飞书任务。
- 同步运行记录。

## 输出

- 网页后台日报。
- 手机端日报。
- 飞书多维表格记录。
- 飞书群消息。
- `ai_outputs` 审计记录。

## 审核规则

- AI 日报默认可人工确认或驳回。
- 高风险建议只做提醒，不自动执行。
- 所有输出保留输入摘要、生成时间和审批状态。
