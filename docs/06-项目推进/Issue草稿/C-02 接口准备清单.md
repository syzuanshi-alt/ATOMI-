# C-02 接口准备清单

## 背景

当前已有抖音、抖店、独立站、TK、飞书。需要先判断哪些能 接口（API） 接入，哪些先导出或手工同步。

## 负责人角色

C

## 开始日期

2026-06-03

## 截止日期

2026-06-05

## 前置依赖

平台账号信息由项目负责人确认。

## 操作步骤

1. 按平台整理一期目标和数据字段。
2. 标记获取方式：接口（API）、导出、手工、半自动。
3. 标记风险等级和账号准备人。
4. 写最晚准备日期。
5. 同步到接口权限表。

## 验收标准

- 每个平台接入路径清楚。
- 不承诺暂时没有权限的平台。
- 可以支持第 1-2 周用测试数据先跑。

## 测试方式

用清单逐项检查账号、权限、字段是否齐全。

## 是否需要 Claude 审核

是，重点检查承诺是否过度。

## 是否客户可见

否。
