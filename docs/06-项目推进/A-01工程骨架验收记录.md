## 常见英文词说明

- pnpm：包管理工具，用来安装和运行项目依赖。
- Next.js：网页应用框架，用来开发电脑端后台和手机端 H5/PWA。
- Prisma：数据库工具，用来管理数据库模型和生成数据库访问代码。
- TypeScript：带类型检查的 JavaScript，用来提前发现代码错误。
- build：构建，意思是把开发代码打包成可发布版本。
- typecheck：类型检查，意思是提前检查代码字段、函数和数据类型是否写错。

## 验收步骤

1. 在 PowerShell 执行：

```powershell
cd D:\SHKF
pnpm install
```

2. 检查数据库结构：

```powershell
pnpm db:generate
pnpm db:validate
```

3. 检查类型：

```powershell
pnpm typecheck
```

4. 启动后台：

```powershell
pnpm dev
```

5. 浏览器打开：

```text
http://localhost:3000
```
