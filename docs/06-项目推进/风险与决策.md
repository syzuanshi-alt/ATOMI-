# 风险与决策

## 当前关键风险

- 平台接口权限审核慢。
- 多租户隔离设计不足。
- AI 输出误执行高风险动作。
- 任务散落在聊天记录里。
- 手机端过早承载复杂后台。
- 飞书被误用为主数据库。

## 已锁定决策

- GitHub 是代码和任务主控。
- Obsidian 只做知识库。
- 飞书只做业务协同出口。
- 一期先做内部试点。
- 手机端先做 手机网页/可安装网页应用（H5/PWA）。
- AI 工具不允许绕过人工审核和 拉取请求（PR）。
