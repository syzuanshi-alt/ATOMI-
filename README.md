# AI 电商自动化平台

面向电商客户的 AI 业务自动化与经营数据中台项目。

## 项目定位

本项目用于构建一套可商用交付的电商 AI 自动化系统，统一抖音、抖店、独立站、TikTok/TK、飞书、邮件、客服、订单和物流数据，支持电脑端后台、手机端 H5/PWA、飞书协同和 AI 日报。

## 协作方式

- GitHub：代码、Issue、PR、节点审核、版本记录的主控中心。
- Obsidian：打开 `docs/` 作为知识库，记录业务逻辑、架构决策、接口清单、数据库模型、风险和节点审核。
- 飞书：业务团队协同出口，不替代 GitHub 和主数据库。
- AI 工具：辅助文档、审查、测试和拆分任务，不自动合并 PR，不直接执行高风险动作。
